## Overview

Serve513 is a community engagement platform designed to connect volunteers with local organizations. The project required creating an intuitive, scalable interface that could serve both individual volunteers and organization administrators.

## Approach

I focused on building a component-based UI system that could scale as the platform grew. Starting with user research and persona mapping, I designed dashboards that surfaced the most relevant information for each user type.

## Key Design Decisions

- **Modular component system** for consistent UI across the platform
- **Role-based dashboards** tailored for volunteers vs. organization admins
- **Intuitive navigation** with clear wayfinding and progressive disclosure
- **Accessible design** following WCAG guidelines for community inclusivity

## Outcome

The component-based system enabled rapid feature development while maintaining visual consistency. User testing showed high task-completion rates for both volunteer sign-ups and organization event management.
